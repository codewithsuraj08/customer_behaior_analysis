Customer Behavior Analysis
Overview

This project focuses on analyzing customer purchasing behavior to generate business insights. The workflow includes data loading, cleaning, exploratory data analysis (EDA), SQL-based querying, and building an interactive dashboard. The final insights are presented through a report and presentation.

Dataset

The dataset contains customer purchase information such as product categories, items purchased, and transaction details. It is used to explore buying patterns, popular products, and category-level trends.

Tools & Technologies

Python – Data loading, cleaning, and EDA

PostgreSQL – Querying and analyzing structured data using SQL

Power BI – Building interactive dashboards and visualizations

Gamma – Creating presentation slides

Jupyter Notebook – Running Python analysis

Project Steps

Data Loading – Imported the dataset into Python.

Data Cleaning – Handled missing values and formatted the data.

Exploratory Data Analysis (EDA) – Explored patterns and distributions.

SQL Analysis – Loaded data into PostgreSQL and ran analytical queries.

Dashboard Creation – Built a Power BI dashboard to visualize insights.

Reporting – Created a report and presentation using Gamma.

Dashboard

Results & Insights

The analysis reveals key trends in customer purchasing behavior, including the most popular products and categories. These insights can help businesses improve marketing strategies and product planning.

How to Run

Clone the repository.

Load the dataset using Python.

Perform data cleaning and EDA in the notebook.

Import the cleaned data into PostgreSQL.

Run SQL queries for analysis.

Open the Power BI file to explore the dashboard.
